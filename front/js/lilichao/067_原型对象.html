Course Summary

in 检查对象属性

使用 in 检查对象中是否含有某个属性时，如果对象中有，或者对象的原型链中有，也会返回 true

hasOwnProperty 检查对象属性

使用 hasOwnProperty 检查对象中是否含有某个属性时，如果对象中有，返回 true；如果对象的原型链中有，返回 false

对象查找属性

  • 首先在自身对象的属性中找
  • 如果没有，会去原型对象 prototype 中寻找
  • 如果没有，会去原型对象 prototype 的原型对象 prototype 中寻找，直到 Object 对象的原型
  • Object 对象没有原型对象(Object 的 __proto__ == null)，如果没有找到，则返回 undefined
