JS 的数据类型

基本值类型

  • string(判断类型: typeof a === "string")
  • number(判断类型: typeof a === "number")
  • boolean(判断类型: typeof a === "boolean")
  • null(判断类型: a === null)(因为 typeof null === "object" )
  • undefined (判断类型: a === undefined 或者 typeof a === 'undefined')

引用类型

  • 普通对象(判断类型: a instanceof Object 或者 typeof c1 === "object"(一般不这么判断，因为区分不了普通的 Object 和 array))
  • 方法（一种特别的对象，可以执行之）（判断类型: a instanceof Function 或者 typeof a === 'function'）
  • 数组（一种特别的对象，数组下标）（判断类型: a instanceof Array）

判断数据类型

  • typeof（返回数据类型的字符串表达，不能区分 null, array, 一般也不用来区分普通 object，其实是可以的）
  • instanceof（判断对象的具体类型）
  • ===（不会进行类型的转换）
