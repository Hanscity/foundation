如何配置 Easy LESS 中，less 和 CSS 的关联地图

1. Extension -> Easy LESS -> { "less.compile": { "compress": false, // true => remove surplus whitespace "sourceMap": true, // true => generate source maps (.css.map files) "out": true // false => DON'T output .css files (overridable per-file, see below) } }

2. ctrl + ,(setting) -> Extension -> Easy LESS Configuration -> Edit Setting Configuration
